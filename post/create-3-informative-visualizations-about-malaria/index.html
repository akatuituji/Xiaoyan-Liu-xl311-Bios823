Create 3 informative visualizations about malaria

2020-09-21 2 min read

There are three datasets in this project. We need to use different methods to visualize each datasets according to their features. We will use seabon and plotnine to do the visualizations.

1. malaria_deaths

Let’s see, there are four columns and they are Entity, Code, Year and Deaths. Since Entity and Code represent the same thing, we just need to choose one feature in our graphs. So we choose Code because it is short and convenient to show in graph. Since Year is continuous, we set it as x-axis, and use Deaths column as y-axis. We use plotnine package and divide all the data into six parts by the number of deaths.






2. malaria_deaths_age：


x-axis: Year/age group, y-axis: Deaths. Because there are 30780 rows, we will not set entity as x-axis. Instead we choose age and year to visualize and use seabon package. There are two graphs, one is for Year and the other is for age.

3. malaria_inc：


We choose seabon package. We set x-axis: Year/Code, y-axis: Incidence. We will have 2 graphs.
